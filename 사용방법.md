# 사용방법

## 가장 쉬운 사용법
이 프로그램은 `변환할PDF/` 폴더에 파일을 넣고 실행 파일만 누르면 됩니다.

변환 결과:
- Markdown: `변환할PDF/변환/<형식>/`
- 원본 이동: `변환할PDF/원본완료/<형식>/`
- 로그: `변환할PDF/logs/conversions.jsonl`

예:
- `sample.pdf` -> `변환할PDF/변환/pdf/sample.md`
- `sample.pdf` -> `변환할PDF/원본완료/pdf/sample.pdf`

## 1회 설치
### macOS
1. repo를 엽니다.
2. `실행/macOS/setup.command`를 더블클릭합니다.
3. 설치가 끝나면 `변환할PDF/` 폴더가 자동 생성됩니다.

### Windows
1. repo를 엽니다.
2. PowerShell에서 `.\실행\Windows\setup.ps1`를 실행합니다.
3. 설치가 끝나면 `변환할PDF\` 폴더가 자동 생성됩니다.

## 매번 쓰는 방법
### macOS
1. `변환할PDF/` 폴더에 PDF, DOCX, PPTX, XLSX, HTML, TXT, CSV, JSON, XML 파일을 넣습니다.
2. `실행/macOS/open_gui.command`를 더블클릭합니다.
3. 변환이 끝나면 결과가 `변환/`과 `원본완료/` 아래에 정리됩니다.

### Windows
1. `변환할PDF\` 폴더에 PDF, DOCX, PPTX, XLSX, HTML, TXT, CSV, JSON, XML 파일을 넣습니다.
2. `실행\Windows\open_gui.bat`를 더블클릭합니다.
3. 변환이 끝나면 결과가 `변환\`과 `원본완료\` 아래에 정리됩니다.

## 다른 폴더 파일을 직접 고르고 싶을 때
기본 방식은 `변환할PDF/` 자동 변환입니다.

직접 파일 선택이 필요하면:
- macOS / Windows 공통: `python 실행/mark_down_gui.py`

이 GUI에서는 원하는 파일을 직접 골라 변환할 수 있습니다.

## 안 될 때
- Python 3.10+가 없으면 먼저 설치해야 합니다.
- `MarkItDown is not installed`가 나오면 setup을 다시 실행합니다.
- 변환할 파일이 없으면 `변환할PDF/` 안에 파일을 먼저 넣어야 합니다.
- `변환할PDF/README.txt`는 안내 파일이라 자동 변환하지 않습니다.
